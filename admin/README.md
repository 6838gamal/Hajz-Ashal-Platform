# 📁 مجلد الإدارة — متابعة سير العمل

هذا المجلد مخصص لك أنت بصفتك مالك المشروع لمتابعة تقدم البناء وخارطة الطريق.

## الملفات

| الملف | الغرض |
|-------|--------|
| `progress.md` | حالة كل موديول (مكتمل / قيد البناء / لم يبدأ) |
| `roadmap.md`  | خارطة الطريق الكاملة مرتّبة حسب الأولوية |
| `decisions.md`| القرارات المعمارية والتقنية المهمة |

## طريقة التحديث

بعد كل جلسة عمل، اطلب من المساعد تحديث `progress.md` ليعكس ما تم إنجازه.
